# Operating-Systems-----Assignment-2

The repository contains the following files:
1. bankers.c - Contains the implementation for the Banker's Algorithm.
2. input.txt - The input file used for the Banker's Algorithm program.

Usage Instructions:
Before any compilation, make sure you are working in a Linux environment. For example, this project was tested on the Kent State University hornet server. Once you get into your Linux environment, you need to create both the bankers and input files by doing: 

- touch producer.c

- touch consumer.c

After creating your files, copy the provided code/text into each respective file.
